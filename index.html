Features and System Requirements

Windows

  • Windows Wasapi Loopback support.
  • ASIO driver is not supported at this time.
  • Windows 10 version 17763.0 or higher.
  • Minimum 2 GB memory recommended.
  • Graphics card recommended.
  • Works with X86, X64, ARM64 processors.

Mac Os

  • Supports Loopback Virtual Audio devices.
  • Absence of Loopback devices, any audio producing app can be used including browsers.
  • macOS 15.1 or higher.
  • Minimum 2 GB memory recommended.
  • Works with Intel, Apple Slicon processors.
